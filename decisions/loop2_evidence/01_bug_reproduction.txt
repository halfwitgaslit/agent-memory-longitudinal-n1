## Diagnosis
1. Mem0's mem0/memory/main.py:747 catches LLMException and logs at error level
2. Mem0's add() returns a successful-shaped result with empty extracted-fact list
3. Our backend's add() sees a "successful" call → records `_record_op("add", error=False)`
4. Health: `_count_memories(uid)` calls `get_all()` which throws → except returns -1
5. Net: `n_errors=0`, `n_memories=-1`, `healthy=True` — silent failure

## Fix plan
1. After mem0.add() returns, if zero results AND zero ids → treat as failure
2. Replace sentinel `-1` from `_count_memories` with: raise OR record error+return last_known
3. Add `last_error` field to BackendHealth and surface in inspect()
4. inspect()['healthy'] must be False if last operation extracted 0 facts unexpectedly
